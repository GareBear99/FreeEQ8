# DAFx26 Demo Track Submission

This folder contains the **official DAFx26 template-formatted** version of the FreeEQ8/ProEQ8 paper for submission to the DAFx26 Demo Track.

## Files

- `DAFx26_FreeEQ8_SUBMIT.tex` — Main LaTeX source (uses official DAFx26 demo template)
- `dafx26demo.sty` — Official DAFx26 demo style file
- `IEEEtranDAFx.bst` — IEEE bibliography style for DAFx

## Building the PDF

### Option 1: Overleaf (Recommended)
1. Go to https://overleaf.com
2. Create a new blank project
3. Upload all files from this folder
4. Click "Recompile"
5. Download the PDF

### Option 2: Local Compilation
```bash
pdflatex DAFx26_FreeEQ8_SUBMIT.tex
pdflatex DAFx26_FreeEQ8_SUBMIT.tex
```

(Run twice to resolve references)

## Submission

- **Portal**: https://dafx26.mit.edu/call-for-demonstrations/ → EasyChair
- **Track**: Demonstrations (2-4 pages, camera-ready)
- **Deadline**: May 22, 2026 (check for extensions)

## Differences from `paper/DAFx26_FreeEQ8.tex`

The original `DAFx26_FreeEQ8.tex` uses a custom template. This version uses the **official DAFx26 LaTeX Demo template** (`dafx26demo.sty`) required for submission.

Content is identical; only formatting differs.
