**Description:**

FreeEQ8 is a free and open-source JUCE/C++ EQ plugin for producers, engineers, and audio developers.

## Release readiness checklist before major launch

- GitHub README screenshots render correctly.
- Release tag exists.
- Builds/installers are attached or build steps are very clear.
- Known gaps are honest and visible.
- License is clear.
- `docs/FEATURED_ON.md` and `docs/SUBMISSIONS_2026.md` are current.
- At least one short demo clip or screenshot is available.
- No claims imply fully validated commercial-grade behavior unless tested in multiple DAWs.
